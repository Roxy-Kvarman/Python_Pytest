# Python_Pytest
Python_Pytest is the test automation project created using Python and Pytest framework.

## Description
The project is built with:
* Design Pattern - Page Model Object
* Selenium WebDriver to interact with the browser (Chrome)
* Python
* Pytest framework
* PyCharm IDE

The project includes the following modules:
- Config with Test data
- Pages 
- Tests with conftest file
- Utilities with Logger configuration and custom assert method
- Logs with log file 
- Reports with pytest html report file\
Log file and html report file contain information about test steps,\
successfull result or failure reason (including uncaught exceptions).

## Usage
The project is ready to run after code downloading.\
One of the tests is made as failed on purpose for the demonstration.
